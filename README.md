# Data Structures and Algorithms Reference

This repository serves as a public reference document and self-instruction guide for common data structures and algorithms frequently asked in technical interviews.

For each data structure and algorithm, you will find:
1. **Source Code**: Python implementations located in the `src/` directory of each topic's folder.
2. **Explainer Notebooks**: Jupyter Notebooks in the `explainer-notebook/` directory that break down the implementation step-by-step, explain the theory, and provide a comprehensive Big O time and space complexity summary table.
